# 谷歌在 Pixel 设备中默认关闭人脸修饰

> 原文：<https://www.xda-developers.com/google-turning-off-face-retouching-pixel-devices/>

在社交媒体时代，自拍已经成为一个历史悠久的传统，但我们对自拍的痴迷实际上可能会对我们的幸福产生负面影响——尤其是在使用滤镜的时候。这就是为什么谷歌在其 Pixel 设备的谷歌相机应用程序中默认关闭了面部修饰算法。这些变化将首先出现在[像素 4a](https://forum.xda-developers.com/pixel-4a) 、[像素 4a 5G](https://forum.xda-developers.com/pixel-4a-5g) 和[像素 5](https://forum.xda-developers.com/pixel-5) 上。

谷歌表示，它是在进行了多项研究并与儿童和心理健康专家交谈后做出决定的。显然，当用户不知道相机或照片应用程序应用了滤镜时，照片会通过设定不合理的美丽标准来对精神健康产生负面影响。

除了默认关闭面部修饰滤镜，谷歌还努力避免提及“美”。图标将被中性语言所改变，因此用户可以更好地理解过滤器的作用以及使用它们的潜在后果。“如果你选择使用面部修饰效果，你会看到更多关于每个设置如何应用以及它对你的图像产生了什么变化的信息，”谷歌在一篇博客文章中说。

谷歌在 Pixel 4 中默认开启了面部修饰功能，但自那以后发生了很多变化。谷歌表示:“这些是我们为支持福祉并将你的声音纳入我们的设计流程而采取的许多步骤中的第一步。”

值得注意的是，在发布 Android 11 的第一个稳定版本后，谷歌发布了最新版本的 [Android 的兼容性定义文件](https://www.xda-developers.com/android-11-features-google-forcing-oems/#tagbar-list:~:text=Disallow%20facial%20appearance%20alteration)，该文件详细说明了 OEM 厂商需要实现哪些功能才能满足 Android 11 的兼容性要求。其中一项要求禁止原始设备制造商在图像处理过程中实施面部修饰算法。然而，修饰仍然可以在 OEM 的相机应用程序中完成。希望其他 OEM 厂商效仿谷歌，默认禁用所有面部修饰功能。
