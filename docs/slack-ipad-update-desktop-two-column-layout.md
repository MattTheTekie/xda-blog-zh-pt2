# iPad 上的 Slack 应用程序接受桌面风格的两列布局

> 原文：<https://www.xda-developers.com/slack-ipad-update-desktop-two-column-layout/>

Slack 刚刚发布了其 [iPad](https://www.xda-developers.com/best-ipad/) 应用的新更新，带来了几个新功能，包括桌面风格的布局、更新的侧边栏和可访问性的改进。

最大和最显著的变化是新的两栏布局，这使 Slack iPad 应用程序更接近其桌面版。当你点击侧边栏中的一个松弛频道或直接消息时，它会在一个单独的右侧窗格中打开，就像 [Mac 应用](https://www.xda-developers.com/best-apps-apple-silicon/)的行为一样。

“iPad 是一款独一无二的设备，它既需要桌面体验的强大功能，又需要移动应用的简单性。Slack Group 产品经理阿克谢·巴克西表示:“我们更新的 iPad 应用程序弥合了这一差距，让用户在一个随时随地工作的世界中保持高效、有条理并与他们的数字总部保持联系。”。

此外，左侧栏也获得了一些增强。现在，您可以折叠部分以隐藏群组和频道，并按最近时间对频道活动进行排序，这样您就可以快速接收新消息和更新。此外，您还可以长按某个频道来调出上下文菜单，以静音该频道、复制该频道的链接或将其标记为已读。

*   部分现在是可折叠的，因此您可以隐藏频道组，只关注最相关的对话
*   现在，Section preferences 将与桌面版本同步，当您在设备之间切换时，可以保持侧边栏的一致性
*   直接信息中的用户名现在有了头像，更容易扫描
*   渠道活动现在可以按最近时间排序，因此您可以更快地查看团队的最新更新
*   只需长按频道名称，您就可以进入快捷菜单，轻松静音或退出频道、复制链接或将其标记为已读

最后，新的更新还带来了一些可访问性的改进。例如，该应用程序现在为苹果 VoiceOver 屏幕阅读功能提供了改进的标签。与此同时，最近的活动排序将更容易更快地找到新的对话。
