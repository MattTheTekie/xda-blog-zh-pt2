# 三星 AKG Y100、Y500、N200 和 N700NC 在印度上市

> 原文：<https://www.xda-developers.com/samsung-akg-y100-y500-n200-n700nc-headphone-earphone-india-launch/>

3.5 毫米耳机插孔肯定要过时了，如果客户仍然关心耳机插孔和有线耳机，他们就没有什么竞争选择了。蓝牙音频外设，即耳机和头戴式耳机，被吹捧为外接电线的最佳替代品——我们当然可以整天辩论无线耳机是否比有线耳机更好。但是，如果你想在市场上购买一些闪亮的新无线耳机，三星希望你看看除了 T2 Galaxy Buds T3 之外的最新无线音频产品，在印度推出 AKG Y100、AKG Y500、AKG N200 和 AKG N700NC 耳机和耳机。

所有四种型号都将通过三星电子商店、独家经销店和所有顶级零售商销售。它们也将在领先的在线门户网站上提供。

### AKG Y100

AKG Y100 是一款颈带式无线耳机，注重轻巧的设计和清脆干净的声音。这款耳机采用了环境感知技术，允许用户通过激活环境感知模式来接收周围的环境声音。其他功能包括支持 AAC 编解码器、快速充电和多点连接(最多 2 台设备)。AKG Y100 有黑色、蓝色、绿色和粉色四种颜色，价格为 **₹6,699** (约 95 美元)。

### AKG Y500

AKG Y500 是一款入耳式无线耳机，承诺长达 33 小时的播放时间。您还可以获得环境感知、AAC 编解码器支持、快速充电和多点连接等功能。此外，耳机可以检测您何时摘下和戴上它们，并可以根据这些动作自动播放/暂停音乐。AKG Y500 有黑色、蓝色、绿色和粉色可选，价格为 **₹9,999** (约 141 美元)。

### AKG N200 型

AKH N200 是入耳式无线“athleisure”耳机，配有一根纤维电缆和支架，可在您锻炼时将它们牢牢固定在耳朵内。这款耳机声称可以提供长达 8 小时的播放时间，支持环境感知、Apt-X 和 AAC 编解码器、快速充电和多点连接等功能。AKG N200 有黑色、蓝色和绿色三种颜色可供选择，价格为 **₹9,999** (约 141 美元)。

### AKG N700NC M2

AKG N700NC 是三星的旗舰耳机之一，采用自适应降噪技术，并承诺温暖的低音响应和无与伦比的清晰度。耳机的设计允许它们折叠起来，三星还包括一些配件，如航空适配器、便携包和一键遥控的无缠结织物电缆，让你可以更好地使用和护理你的耳机。AKG N700NC 承诺播放 23 小时。这款耳机只有一种黑色，价格很酷 **₹26,999** (约 380 美元)，比同类领先的[索尼 WH-1000XM3](https://www.amazon.in/Sony-WH-1000XM3-Wireless-Cancellation-Headphones/dp/B07HZ8JWCL/?tag=xdaportalin-21) 略贵(尽管你可以在销售期间买到便宜到₹23,490(330 美元)的索尼耳机)。
