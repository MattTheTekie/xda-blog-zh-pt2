# 小米将在本月结束所有设备的 MIUI 全球测试版

> 原文：<https://www.xda-developers.com/xiaomi-ending-miui-global-beta-all-smartphones-july/>

小米的 MIUI 经历了不断的改进，测试版测试人员在保留和增强功能方面发挥了关键作用，然后才传递到稳定版本。MIUI 有两层测试版——一层是封闭测试版，它有一个更不稳定的 ROM，并且仅限于有限数量的测试人员；另一层是公开测试版，它基本上是稳定的，并且拥有几乎可以集成到稳定版本中的功能。但是随着[小米作为一个品牌](https://www.xda-developers.com/oneplus-oppo-xiaomi-fastest-growing-premium/)成长，不仅在亚洲，而且在欧洲等[市场，它打算限制其 beta 测试的努力，并专注于为其庞大的设备目录提供稳定的更新。](https://www.xda-developers.com/chinese-oems-huawei-xiaomi-european-market-share-2018/)

小米通过其社区论坛宣布，从 7 月 1 日开始，它将停止推出全球测试版 rom。帖子指出，由于 MIUI 公测版 rom 大多稳定，用户倾向于将其作为日常驱动程序来使用，以珍惜新功能。然而，这些用户中只有一小部分以修复这些功能所必需的反馈形式回馈给小米。

小米强调，随着其不断增长的设备组合和更新的市场，它希望专注于为用户提供稳定的体验。除了更稳定的体验，这一步骤预计将允许小米更频繁地向 mi 和 Redmi 品牌的设备推送稳定的更新。这也应该有助于 MIUI 作为一个操作系统的发展。

该声明是在该公司之前关于[放弃入门级红米设备](https://www.xda-developers.com/xiaomi-miui-beta-for-budget-redmi-smartphones/)的 MIUI 测试版的声明之后发布的。我们可以从这些公告中解读出，小米用许多类似的智能手机来饱和价格区间的策略可能最终开始让其开发者无法抗拒。除了进入新市场，小米及其分拆品牌 Redmi 一直在探索智能手机的新途径，如 [Redmi K20 Pro](https://www.xda-developers.com/xiaomi-redmi-k20-pro-launch-china/) 。

小米表示，它将继续寻求用户的反馈，但我们不确定这是否(在没有测试版 rom 的情况下)会有利于我们当中的修补者。好消息是，小米还为其大多数设备提供内核源代码，因此你可以随时转向小米智能手机的专用 XDA 论坛，以进一步改善你的软件体验。

我们也希望这一决定将有助于小米继续为其智能手机开发 [MIUI 11](https://www.xda-developers.com/miui-11-new-features/) 和 [Android Q](https://www.xda-developers.com/xiaomi-mi-mix-3-redmi-k20-pro-miui-android-q/) 更新。
