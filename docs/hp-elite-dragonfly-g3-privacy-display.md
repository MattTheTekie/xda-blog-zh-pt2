# 惠普精英蜻蜓 G3 有隐私显示屏吗？

> 原文：<https://www.xda-developers.com/hp-elite-dragonfly-g3-privacy-display/>

企业用户经常不得不处理敏感信息，而且很多时候，他们不得不在公共场合工作。为了帮助保护敏感信息的隐私，一些商用笔记本电脑配备了所谓的隐私显示器，它可以保护你的屏幕免受周围人的影响。随着[惠普精英蜻蜓 G3](https://www.xda-developers.com/hp-elite-dragonfly-g3/) 即将发布，你可能想知道它是否也有隐私显示屏。谢天谢地，是的，这是一个选择。

虽然默认情况下不包括隐私显示屏，但您可以为惠普精英版蜻蜓 G3 添加隐私显示屏。惠普称之为 Sure View Reflect，这款笔记本电脑只有三种显示器配置中的一种。如果你想获得隐私显示，你必须选择全高清+ (1920 x 1280)分辨率选项。顶层的 3K2K 有机发光二极管显示器没有隐私显示器。

## 精英版蜻蜓 G3 的隐私显示屏

HP Sure View Reflect 是惠普最新一代的隐私显示器。公司做这个已经有一段时间了，这一代自然是数一数二的。它的作用是，如果你从一个角度看屏幕，你看不到显示屏上的内容。取而代之的是，屏幕会呈现铜色，这样你周围的人就看不到任何内容了。这样，如果你坐在咖啡店里，你就不用担心其他人会看到这些信息。具体来说，惠普表示，Sure View Reflect 在 45 度角的能见度不到 1.5%。

HP Sure View Reflect 的一个优势是它为用户提供了更好的观看体验。为了向其他人隐藏信息，隐私显示器通常会让用户更难看到屏幕本身，但肯定 View Refect 会好一点。它在户外环境下应该仍然可见，而且惠普表示，与 Sure View 的前几代产品相比，它提供了更好的图像清晰度。具有 Sure View Reflect 的显示器可以达到 1000 尼特的亮度，即使在明亮的条件下也能确保可视性。

虽然它需要特殊的硬件，但 Sure View Reflect 实际上是一个软件功能，所以只有当你需要保护私人信息免受窥探时，才可以启用它，而不会在你不使用它时影响观看体验。

* * *

除了隐私功能，惠普精英版蜻蜓 G3 的显示屏也是一个很好的体验。它的长宽比为 3:2，明显高于之前型号的 16:9 显示屏。更高的屏幕也有助于提高工作效率，因为你不用滚动屏幕就可以一次看到更多的内容。隐私显示选项有全高清分辨率，但如果你不需要该功能，你可以升级到 3K2K (3000 x 2000) OLED 面板，它会更清晰，看起来很棒。有机发光二极管显示器提供鲜艳的色彩、真实的黑色和非常高的对比度，使其非常适合媒体消费。
