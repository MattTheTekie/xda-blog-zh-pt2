# 主题 MIUI 动画时钟

> 原文：<https://www.xda-developers.com/themed-miui-animated-clock/>

XDA 成员 [jamieaxall](http://forum.xda-developers.com/member.php?u=627664) 的 MIUI 模拟时钟模块的粉丝们，你们可能会喜欢 XDA 成员 [00McD00](http://forum.xda-developers.com/member.php?u=2761201) 创作的主题。

这位 XDA 成员在截屏主题中贴了几张这样的图片，看到下载量很大，所以决定用所有的颜色做一个单独的主题。主题 MIUI 动画时钟安装就像任何其他应用程序，然后你可以在你的其他部件中找到它。

看起来这位 XDA 会员也在接受在背景中有运动队标志和其他标志的时钟的请求，所以如果你感兴趣，请前往查看[主题线索](http://forum.xda-developers.com/showthread.php?t=1053899)。
