微软的Clipchamp视频编辑器现在可以让你免费制作1080p的视频

原文：https://www.xda-developers.com/microsoft-clipchamp-1080p-video-free/#0001-01-01

Windows 11中包含的视频编辑器clipcamp在首次推出时表现平平，现在得到了一个早该得到的改进。微软正在将对1080p视频导出的支持引入Clipchamp的免费层，这意味着你不再需要为一个相对基本的功能支付额外费用。

Clipchamp是一个基于云的视频编辑器，微软去年在Windows 11发布之前收购了它。发布时，Windows 11附带了一个Clipchamp PWA的快捷方式，必须下载并安装，但微软最近开始在操作系统中预装该应用。这时，一些用户注意到，考虑到免费层只允许用户以少得可怜的480p分辨率导出视频，定价结构似乎是一种敲诈。即使是每月9美元的第一付费层，也只能给你720便士的出口。

现在，1080p视频导出一直包括到免费层。付费层的所有其他额外待遇都是一样的，其中包括创作者层的无限云存储和音频库存(每月9美元)，商业层的品牌套件(每月19美元)，以及商业高级层的无限视频和图像库存(每月39美元)。奇怪的是，微软仍然不允许用户在任何Clipchamp层上以高于1080p的分辨率导出视频。如果你对4K视频感兴趣，这仍然是不理想的。

尽管如此，这是一个非常必要的变化，一个有趣的考虑到微软说，它不会改变clipcamp的定价结构。公平地说，价格和以前一样，但随着每个人都可以使用1080p视频，价值主张发生了显著变化。未来，新的价值是否会被添加到更高的层级，或者微软是否会简化定价结构，还有待观察。

将额外的功能捆绑到微软365上是有意义的，但没有官方迹象表明会发生这种情况。我们已经就这些变化联系了微软，如果有回音，我们会更新这篇文章。
