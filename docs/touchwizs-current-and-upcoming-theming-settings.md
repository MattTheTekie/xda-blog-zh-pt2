# TouchWiz 的外观以及如何改变它

> 原文：<https://www.xda-developers.com/touchwizs-current-and-upcoming-theming-settings/>

TouchWiz 是迄今为止最有争议的 ROM，以使其成为旗舰。自 S 系列早期以来，人们每天都将它与股票 Android 和竞争对手的产品进行比较。虽然三星确实经常偏离谷歌最初和个人对 Android 的愿景，但事实证明，其中一些偏离不仅对三星的收入，而且对整个 Android 平台都是成功的。我们每天使用的功能，如快速切换，最初是由三星推广的，我怀疑像扩展的电源菜单和多窗口这样的东西可能很快也会来到 AOSP，因为它们似乎早就应该出现了。

但是，随着 TouchWiz 过去所做的每一件正确的事情，他们也犯了同样糟糕的错误。三星在其软件中出现的许多小错误、小故障或不一致之处都可以被忽视，尽管推出的许多功能并没有真正增加什么，但也不一定会从用户体验中减少什么...对吗？嗯，可能不是这样。疯狂地添加一些不是噱头就是毫无用处的特性，特别是那些执行起来有缺陷但代码相当复杂的特性，比其他竞争者添加的许多特性对用户体验的影响更大。事实上，他们在两个主要方面做到了，而且(尤其是)做到了:

## **性能**

TouchWiz 一直被认为是最笨重的 rom 之一，不仅充斥着没人要的功能和应用程序，而且[似乎没人使用](http://www.androidcentral.com/study-shows-samsung-users-spend-little-time-bloatware-apps)。设备不仅仅占用设备内部存储器的空间，许多应用程序或模块不能被卸载；它们还会占用大量的内存，有些会永久缓存在内存中，如果不启动设备，就无法关闭服务或禁用应用程序。这是一个问题。当还考虑到 TouchWiz 启动器似乎是它们中最落后的，并且系统已经占用了比正常情况下更多的内存来容纳 ROM 支持的所有功能时，我们可以看到为什么 TouchWIz 在过去表现如此差的原因之一。不过还是有希望的，因为早期的用户报告和测试已经证明，TouchWiz 的一些 Lollipop ROMs 在这些方面有了显著的改进。你可以在 XDA 上了解更多关于[的信息。](http://www.xda-developers.com/android/analysis-of-the-galaxy-note-3-android-5-0-leak/)

## **设计**

这么说吧:三星的设计一团糟，而且不仅仅是美学上的。在过去的 3 年里，我已经使用了 3 台三星设备，比起它们，我一直更喜欢我的库存设备(Nexus 5，Nexus 7)。在很多情况下，我发现自己就是因为这个原因而完全摆脱了 TouchWiz。现在，TouchWiz 的旧外观在过去几年中并没有真正看到多少偏差或改进，没有足够的实质性变化来摆脱它所积累的污名。虽然这种外观似乎受到韩国观众的称赞，但至少可以说，对西方用户来说，他们的选择是冲突的。颜色看起来并不令人愉快，线条大多很清晰，一些地方有不必要的渐变和原始的黑色背景，虽然对 AMOLED 屏幕来说很好，但与其他项目形成了暗淡的对比。简而言之，与一些更优雅或更全面的 rom 相比，TouchWiz 过去是，现在仍然是丑陋的。在设计的功能方面，它的缺点是陈旧和无意中混淆了抽屉项目、设置和功能的组织。

## **绕过它**

Android 的美妙之处在于，如果你不喜欢某样东西，你可以随心所欲地调整它。如果你不喜欢默认的消息应用，你可以使用 Hangouts。如果你不喜欢默认的启动器，你可以使用 Nova。如果你不喜欢 ROM 本身，你也可以改变。我想如果你在 XDA，你可能已经非常了解这一点。但有时，在 OEM ROMs 上进行更改并不容易。而有些东西是无法完全改变的，或者说是无法一直改变的。三星手机就像这样，虽然你可以通过定制恢复来轻松地将不同的主题应用到你的设备上，但这些主题在美学、设计或功能上的差异却缺乏自由。

避免用新的只读存储器来改变主题的三星用户有其他的选择。一个非常流行的选择是使用[exposed Framework](http://forum.xda-developers.com/xposed)来调整一些东西，比如通知栏、阴影，甚至多任务处理和设置菜单。虽然改变设计主要限于添加或删除东西，或者改变颜色和透明度，但你最终会得到与 TouchWiz 大不相同甚至更好的美感。下面是我在接触魔法的一年中完成的一些例子。

如果你有兴趣知道如何修改你的设备，你可以在 XDA 上做以下任何一件事:

*   检查之前链接的 [Xposed](http://forum.xda-developers.com/xposed) 框架。如果你有一部三星手机，一些让你的主题化进行的好模块将会是充满扭曲的[重力盒](http://forum.xda-developers.com/xposed/modules/app-gravitybox-v3-1-4-tweak-box-android-t2316070)或[Wanam exposed](http://forum.xda-developers.com/xposed/modules/app-wanam-xposed-customize-stock-t2383484)。还有很多其他模块可以为你的用户界面提供单独的模块，比如之前展示的[模糊系统 UI](http://repo.xposed.info/module/com.serajr.blurred.system.ui) 。
*   只需进入设备的“主题和应用”子论坛，找到一些主题和模式
*   自己动手做 XDA 上的大量文档和指南，比如我们今天早些时候分享的由公认的 XDA 主题编辑的 XML 编辑指南。

## 我还能做些什么来改变我的用户界面的外观？

好吧，看起来如果你不愿意根化你的设备，不愿意安装需要的恢复或框架来充分利用你的主题化的可能性，你也许可以**简单地等待**。几个月前，著名的揭发者 Sammobile 报道 TouchWiz 可能会采用一个主题引擎，就像 CyanogenMod 和其他 rom 一样，尽管其相似性的延伸没有得到证实。早期的截图显示了新的可能主题，但不清楚这是泄漏还是伪造的。

后来，[他们向我们展示了](http://www.sammobile.com/2014/12/04/more-details-about-samsung-touchwiz-themes-explained/)新的主题选项将出现在 A3 和 A5 机型上，但将仅限于系统图标、壁纸和配色方案。这让许多三星用户感到失望，他们期待过时丑陋的默认 ROM 有更多的可定制性。然而，就在几个小时前，当 Sammobile 独家披露未来[三星设备将推出增强主题支持](http://www.sammobile.com/2015/01/07/exclusive-future-samsung-devices-to-support-enhanced-software-themes/)时，希望再次燃起:

“根据我们收到的信息，三星正在让所有未来设备优先支持主题，S6 将是首批提供增强主题定制的设备之一。”

“主题功能将类似于 Galaxy A 设备上看到的功能，将被称为主题中心(或类似的东西)，但有一些额外的选项，如改变系统声音、字体和所谓的事件。”

更重要的是，新设备将拥有一个*主题商店。*虽然我们不确定它是否会有第三方主题，或者开发者可以选择创建自己的主题并上传，但三星以积极发布必要的开发者工具而闻名，以利用他们手机通常具有的独家三星功能，如多窗口或最近的 [Galaxy Note Edge](http://forum.xda-developers.com/note-edge) 的侧显示。因此，我们希望在 Galaxy S6 发布之前看到更多这方面的内容，我们迫不及待地想知道 TouchWiz 内置的第三方主题支持将允许什么。

## 等还是不等？

您现在可以选择定制您的 Samsung 设备。也许三星将为那些想用自己的风格制作自己的 ROM 的人提供绝佳的机会。但鉴于这是三星，我不会抱太大希望...虽然，如果有什么值得承认的是，尽管 TouchWiz 多年来在许多方面都不如竞争对手，但他们似乎正在扭转局面。让我们只希望他们抛弃旧的设计语言，向谷歌和其他公司学习如何制作好看的界面。
