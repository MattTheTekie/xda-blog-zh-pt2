# TCL 推出阿尔卡特 3、3L、1S 和低成本可折叠显示屏

> 原文：<https://www.xda-developers.com/tcl-alcatel-3-3l-1s-low-cost-foldable-displays/>

过去几年，TCL 一直是 MWC 的知名常驻者。通常，他们会发布几款不同的智能手机，其中大多数都是经济型或中档产品。2019 年也不例外。这家中国 OEM 厂商刚刚宣布了阿尔卡特品牌下的三款设备，并展示了其 DragonHinge 可折叠显示屏的原型。让我们来看看。

## 阿尔卡特 1S

这是今年 MWC 阿尔卡特品牌下最便宜的智能手机。Alcatel 1S 是一款经济型智能手机，采用 Unisoc SC9863A 八核片上系统，主频为 1.6 GHz，3 GB 内存，32 GB 内部存储，5.5 英寸高清+显示屏，纵横比为 18:9，电池容量为 3，060 mAh。在手机背面，你会看到一个 1300 万像素+200 万像素的双摄像头系统，以及正下方的指纹扫描仪。前置摄像头镜头也就 5MP。另外值得注意的是，这款智能手机运行的是 Android Pie。

阿尔卡特 1S 将于 2019 年第二季度上市。它有四种颜色:黑色、蓝色、玫瑰色和金色，售价仅为€109 英镑。

## 阿尔卡特 3

新宣布的智能手机家族的下一个代表是阿尔卡特 3。这是上一个型号的略强版本，但仍远未达到中档水平。Alcatel 3 配备了功能更强的高通骁龙 439 八核片上系统，主频为 2.0 GHz，3/4 GB 内存，32/64 GB 内部存储，5.94 英寸高清+显示屏，纵横比为 19.5:9，电池容量为 3,500 mAh。前置摄像头凸起至 8MP，后置双摄像头系统配置为 13MP + 5MP。你还会在手机背面看到一个指纹扫描仪。令人惊讶的是，这款机型自带 Android 8.1 奥利奥，但 TCL 承诺他们会在推出后很快更新为 Android Pie。

阿尔卡特 3 将于 2019 年在 Q2 上市，€159 (3/32)和€189 (4/64)有渐变黑蓝和渐变蓝紫两种颜色可供选择。

## 3L 阿尔卡特

我们列表中的最后一款设备是 Alcatel 3 的精简版。3L 阿尔卡特配备了略低端的高通骁龙 429，仍然是四核处理器，主频为 2.0 GHz。它还具有 2 GB 的内存和 16 GB 的内部存储。电池容量、相机系统、显示屏尺寸和软件版本与 Alcatel 3 完全相同。据 TCL 称，该设备上的指纹扫描仪将仅在部分地区可用。

2019 年，3L 阿尔卡特将和它的两个兄弟姐妹一起在 Q2 上市。这款智能手机将有金属蓝和煤黑色两种颜色。

## DragonHinge 可折叠显示屏

TCL 还谈到了他们的 DragonHinge 低成本可折叠显示器概念。在这一点上，它只是一个原型，但中国制造商计划在未来将它们用于真正的智能手机。他们想出了一些可折叠显示器的独特用法，如腕带和其他一些东西。据称，搭载 DragonHinge 可折叠显示屏的平价智能手机将于明年 2020 年上市。
