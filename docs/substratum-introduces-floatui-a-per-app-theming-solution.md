# Substratum 引入了 FloatUI:一个基于应用的主题化解决方案

> 原文：<https://www.xda-developers.com/substratum-introduces-floatui-a-per-app-theming-solution/>

最近，我们爆料说 CyanogenMod 主题引擎[将不会在 linegeos](https://www.xda-developers.com/exclusive-dead-cyanogenmod-theme-engine-cmte-not-coming-back-lineageos-replacement/)中回归。

这一发展对定制主题化的世界是一个暂时的打击，因为 CMTE 多年来一直是主要的主题化框架之一。然而，它并不是唯一一个为你的设备提供主题化功能的框架。Substratum 是 CMTE 的一个受欢迎的替代品，这个项目背后的开发者 T2 最近和我们坐在一起回忆他们过去、现在和未来的工作。

谈到未来的工作，XDA 承认开发人员尼科尔森舒姆(nicholaschum)现在准备开始一个新项目，他和他的团队正在开发这个项目，名为“**float ui(T3)”。该项目基本上可以被描述为一个“**每应用程序主题化解决方案**”，查姆先生正在给我们一个浮动推期待什么的潜行高峰。**

* * *

## 遇见 FloatUI:你的每个应用的主题化解决方案

正如你在上面的视频中看到的，一旦激活，就会有一个小图标一直浮在主屏幕上。因此，无论何时启动应用程序，都可以使用 FloatUI。让我们考虑一个例子:

让我们假设你已经在你的设备上安装了主题[单声道/艺术](https://play.google.com/store/apps/details?id=connoisseurship.project.mono&hl=en)和[幻想](https://play.google.com/store/apps/details?id=com.kohlewrrk.reverie.subs&hl=en)，并且 FloatUI 已经被激活。现在，你前往设置，将设置应用程序的主题更改为单声道/艺术。更改将在一秒钟内完成。你现在走向通知区域，并希望它看起来像一个幻想提供的。使用 FloatUI，通知将立即主题化，与设置页面分开。

因此，您可以为特定的应用程序设置特定的主题**。您还可以切换框架覆盖。此外，主题应用在第二个应用程序中，没有留下任何元素。下面是查姆先生的视频，让我们先来看一看，亲自体验一下 FloatUI。**
