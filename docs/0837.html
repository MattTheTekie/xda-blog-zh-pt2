July 2020 Android security update live for Google Pixel and Galaxy S20

原文：https://www.xda-developers.com/july-2020-android-security-update-google-pixel-samsung-galaxy-s20/#0001-01-01

我们已经进入7月6天了，但由于今天是这个月的第一个星期一，这意味着是时候发布最新一批Android安全补丁了。谷歌发布了7月份最新的Android安全公告，并推出了对Pixel 4、Pixel 4 XL、Pixel 3a、Pixel 3a XL、Pixel 3、Pixel 3 XL、Pixel 2和Pixel 2 XL的更新。三星还推出了一个新的更新，将相同的安全补丁级别(SPL)带到了美国的Galaxy S20系列

与上个月的Android安全更新不同，这次更新没有伴随着Pixel功能的下降。此外，谷歌没有概述Pixel系列的任何“值得注意的修复”。这是一个彻底的安全更新，这个月已经解决了7个问题。像往常一样，漏洞从高到严重，最严重的与我们的老朋友有关:“媒体框架和远程攻击者可能通过特制的文件执行任意代码。”

Along with the Pixel devices mentioned above, Samsung is rolling out the July 2020 Android security update to the Galaxy S20 series. Samsung hasn't shared many details about the update, but the build numbers are G981U1UES1ATFB (S20), G986U1UES1ATFB (S20+), and G988U1UES1ATFB (S20 Ultra) for the U.S. unlocked models.
