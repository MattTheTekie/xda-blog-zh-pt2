# 高通宣布推出骁龙 712，与骁龙 710 相比性能略有提升

高通宣布了该系列的下一款芯片组:骁龙 712。正如你可能从型号上看出的，这是一个小升级。

近一年前，高通骁龙 710 SoC 发布[。700 系列比中端的 600 系列略有提升，但还不到 800 系列的水平。我们已经在来自](https://www.xda-developers.com/qualcomm-snapdragon-710-announcement/) [Oppo](https://www.xda-developers.com/oppo-r17-r17-pro-india/) 、[诺基亚](https://www.xda-developers.com/nokia-x7-qualcomm-snapdragon-710/)以及该领域其他设备的许多设备中看到了这一点。今天，高通宣布了该系列的下一款芯片组:骁龙 712。正如你可能从型号上看出的，这是一个小升级。

骁龙 712 是一款基于 10 纳米的芯片，采用 64 位架构和 8 个 Kyro 360 CPU 内核。就像骁龙 710 一样，712 也有 Adreno 616 GPU 和骁龙 X15 调制解调器。时钟速度是骁龙 712 稍微升级的一个方面。它的时钟速度更高，为 2.3GHz(相比之下为 2.2GHz)。快充 4.0+充电速度也略有提升。

高通还为骁龙 712 增加了一些新的音频技术。TrueWireless Stereo Plus 适用于无线耳塞。广播音频有助于用户将音频同时传输到多个蓝牙设备。正如你在 2019 年所预期的那样，高通还表示，712 提供了更智能、更快速的人工智能集成。712 并不是 710 的主要升级，所以看看哪些设备将推出最新的芯片组将会很有趣。这个月晚些时候，我们可能会在 MWC 听到更多的消息。
