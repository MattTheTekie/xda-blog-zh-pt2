# 荣誉 8X 相机测试

> 原文：<https://www.xda-developers.com/honor-8x-camera-test/>

在这个 XDA 电视视频中，我们将荣誉 8X 进行了终极相机测试。我们将它与 Honor 最近推出的其他手机进行了比较，测试了视频功能、自拍模式、慢镜头以及你能想到的所有其他功能。
