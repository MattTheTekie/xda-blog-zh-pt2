# POCO F1 的最新 LineageOS 16 版本修复了触摸屏问题

# POCO F1 的最新 LineageOS 16 改进了触感，并增加了对红外摄像头的支持

Poco F1 最新的非官方 LineageOS 16 版本实现了小米最新的触摸屏补丁以及对红外摄像头的支持。

小米新子品牌 POCO 的 POCO F1 是一款令人惊叹的手机，因为它为预算紧张的人提供了旗舰体验。由于拥有高通骁龙 845，高达 8 GB 的内存和 UFS 2.1 存储，这款手机的速度非常快，但该设备还具有 4,000mAh 电池，双后置摄像头，可扩展存储，3.5 毫米耳机插孔，6.18 英寸的大型 FHD+显示屏，以及由红外摄像头增强的面部解锁。对于那些不喜欢小米 MIUI 软件的人来说，POCO 鼓励你通过闪存定制 ROM 来掌握自己的事情。定制 ROM 可能会引入一些普通 ROM 中不存在的错误，但普通 ROM 中也可能会有一些错误被带到定制 ROM 中。幸运的是，两个常见的错误现在都有了补丁。

首先，我们已经看到来自小米 POCO F1 [用户的报告，这些用户受到](https://piunikaweb.com/2018/09/15/xiaomi-offers-update-on-poco-f1-touch-problem-and-other-issues/)各种触摸屏问题的困扰，如鬼影触摸或多点触摸问题。幸运的是，小米已经开发了一种可以改善触摸的补丁。这些补丁在最新的全球开发者 ROM MIUI 8 . 11 . 23 中提供。然而，如果 MIUI 不是你的那杯茶，那么你可能会很高兴知道这个修复也已经进入了定制 rom，因为[最新的非官方 LineageOS 16 build](https://forum.xda-developers.com/showpost.php?p=78270629&postcount=1944) (基于 Android Pie)已经包括了这些补丁。此外，最新的 LineageOS 16 版本还增加了对手机前置红外摄像头的支持，该摄像头用于增强手机在弱光条件下基于软件的人脸检测。POCO F1 的面部检测不如 Face ID 等其他面部扫描解决方案安全，但红外摄像头为这一考验增加了一点便利。随着新补丁集成到非官方的 LineageOS 16 版本中，红外摄像头现在可以用于在弱光条件下增强谷歌智能锁的面部解锁功能。
