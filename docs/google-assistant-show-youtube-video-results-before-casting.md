# 你现在可以要求谷歌助手在选角前展示 YouTube 视频结果

> 原文：<https://www.xda-developers.com/google-assistant-show-youtube-video-results-before-casting/>

谷歌助手很智能，而且每次更新都会变得更智能。它可以为你做很多事情，从设置闹钟到滚动魔法球，甚至通过 Nest 或飞利浦 Hue 等设备控制你的家。虽然它可以为您做所有这些事情，但有时它缺乏的功能会让用户感到惊讶。这变得更加奇怪，因为谷歌似乎增加了一些功能，有些人认为这些功能比其他人更重要，比如可以设置你自己的音频，以便在助手设置的闹钟中播放。直到最近，谷歌才悄悄地通过谷歌助手在 Chromecast 上添加了搜索 YouTube 视频的支持，这似乎也是很久以前就应该添加的东西。

要激活该功能，只需让任何已启用设备上的虚拟助手在您支持 Chromecast 的电视上显示某一类型的视频列表。如果你的电视被命名为“卧室电视”，只需说“嘿，谷歌，在我的卧室电视上列出 YouTube 上的 x 个视频”，其中 x 是你可以想象的 YouTube 上的任何内容。你也可以说“好吧，谷歌，在我卧室的电视上给我看看 YouTube 上的 x 个视频。”它使用简单，可以大大增强您的家庭影院系统。

https://www.youtube.com/watch?v=-4mRlti2sA0

来自/u/otaviosk 的一段视频展示了 Chromecast 电视上的新功能。

该功能似乎正在慢慢推出，因为多个用户报告无法使用该功能。不过，我们的编辑米莎尔·拉赫曼可以证实这是可行的。这是未知的，如果它将是一个地区的独家，或后来绑定到一个用户的帐户，有一个 YouTube 红色订阅或没有。

似乎没有办法强制启用该特性，并且似乎有一些限制。首先，你不能从你的订阅中播放视频，其次，它似乎不会用于第一代 Chromecast。不过，你可以让它列出某个 YouTuber 上的视频。例如，简单地说“嘿，谷歌，在 YouTube 上列出 XDADeveloper 的视频”，你就会在我们的 YouTube 频道上获得一系列最新的视频。尝试一下，并让我们知道它是否在您的设备上工作！
