# 出色的 Android 天气应用/动态壁纸/小工具

> 原文：<https://www.xda-developers.com/yowindow-mobile-weather-app/>

很难反驳历史事实。其中一个与移动设备有关的问题是，在 HTC Vogue 时代，随着 TouchFlo for Windows Mobile 的推出，移动天气预报的面貌发生了巨大变化。UI 基本上向我们介绍了我们今天所知道的主屏幕上的小部件。神话般的用户界面中的一个标签显示了一个漂亮的、简洁优雅的天气预报，伴随着好看的天气状况图像。

随着时间的推移，TouchFlo 演变成了马尼拉——后来，演变成了我们今天所知道的 Sense。这不仅包括前面提到的图形，还在你眼前添加了天气动画。(谁能忘记雨滴落在你屏幕上时的雨刷效果？)

从那时起，凭借更好的硬件和更多使用 Android 做更多事情的自由，天气应用程序逐渐成为我们可以期待在移动设备中拥有的一部分。令所有人沮丧的是，如今大多数制造商提供的某种天气和时钟小工具都不如 Sense 在过去的统治时期所提供的。诚然，有许多第三方的替代品，但就完整的软件包而言，XDA 论坛成员 [par128](http://forum.xda-developers.com/member.php?u=5956306) 的 YoWindow Weather 独占鳌头。

YoWindow Weather 是一个一体化的解决方案，适用于那些对 TouchWiz 平淡无奇的 home widget 不满意的人，那些想要远离 Sense 的人，或者甚至是那些没有花哨的原生天气应用程序的 AOSP 用户。该应用程序不仅可以作为一个独立的应用程序，也可以作为一个美丽的动态壁纸，描绘全球不同城市的天气和时间。LWP 是完全动画化和交互式的，让您可以看到正在发生的事情和一天中稍后将会发生的事情。白天和夜晚之间的时间转换，你还可以在屏幕上添加小部件来扩展预测选项。

该应用程序是开发人员的第一次伟大努力，他正在寻求反馈，以尝试和改进这种天气解决方案。你有什么建议可以和开发者分享吗？如果你这样做了，请停下来，留下你的反馈或者你碰巧发现的错误。更多信息可以在 [YoWindow 天气的原帖](http://forum.xda-developers.com/android/apps-games/app-yowindow-weather-beta-t2823108#post54515282)中找到。
