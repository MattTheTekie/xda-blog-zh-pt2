# 这些三星设备将在一个 UI 4.1 中接收新的相机功能

三星分享了所有旧 Galaxy 旗舰产品的列表，这些产品将通过一个 UI 4.1 获得对新相机功能的支持。请继续阅读，了解更多信息。

上个月，三星发布了基于安卓 12(T1)的新版用户界面，其旗舰产品是 T2 Galaxy S22(T3)。软件版本[包括一系列新的相机功能](https://www.xda-developers.com/samsung-galaxy-s22-camera-features/)，包括带宠物识别的高级人像模式，支持夜间模式的人像拍摄，支持人像视频的长焦镜头，等等。在过去的几天里，三星[向几款老款 Galaxy 智能手机推出了 One UI 4.1 更新](https://www.xda-developers.com/samsung-galaxy-note-20-galaxy-s21-fe-one-ui-4-1/)，为这些设备带来了一些新功能。然而，该公司并不打算将所有新的 One UI 4.1 相机功能推广到旧的 Galaxy 旗舰产品上。

根据三星社区论坛上最近的一篇帖子，用于旧 Galaxy 旗舰机的 One UI 4.1 更新将包括一些新的 Galaxy S22 相机功能。但更新不会给所有设备带来所有新功能。查看以下部分，了解您的设备将提供哪些相机功能的更多信息。

*   **夜间模式人像:** Galaxy S21 系列、Z Fold 3、Z Flip 3、S20 系列、Note 20 系列、Z Fold 2、Z Flip 5G
*   **宠物识别:**银河 S21 系列，Z Fold 3，Z Flip 3，S21 FE，S20 系列，Note 20 系列，Z Fold 2，Z Flip 5G，S20 FE
*   **人像模式光线定位:** Galaxy S21 系列、Z Fold 3、Z Flip 3、S21 FE、S20 系列、Note 20 系列、Z Fold 2、Z Flip 5G、S20 FE
*   **长焦镜头人像视频:** Galaxy S21 系列，Z Fold 3，S21 FE
*   **增强导演视角:** Galaxy S21 系列，Z Fold 3，Z Flip 3
*   **长焦 pro/pro 视频支持:**Galaxy S21 Ultra；Galaxy Z Fold 3、Note 20 Ultra、S20 Ultra 和 Z Fold 2(计划于 2022 年上半年支持)
*   **Snapchat 集成:** Galaxy S21 系列

此外，帖子透露，One UI 4.1 还将为这些设备上的相机 UI 带来一些变化。例如，单次拍摄模式将被移动到“更多”菜单，AR 涂鸦模式将被移动到“AR 区域”菜单。此外，该公司正在计划其他改进，如支持使用长焦镜头的夜间人像摄影，使用人工智能的高分辨率图像的新的详细增强功能，以及自动取景功能的改进。这些改进将在今年上半年推广到 Galaxy 智能手机上。
