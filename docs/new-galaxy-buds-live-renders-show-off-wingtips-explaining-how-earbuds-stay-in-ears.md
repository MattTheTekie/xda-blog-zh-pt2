# 新的银河芽现场渲染炫耀翼尖，解释如何耳塞将留在你的耳朵里

三星 Galaxy Buds Live 的新渲染图从侧面展示了即将到来的 TWS，突出了菜豆形耳塞上的翼尖。

距离三星 Unpacked 2020 还有[一周](https://www.xda-developers.com/samsung-galaxy-note-20-galaxy-unpacked-august-5/)，我们继续收到更多关于预计将在今年下半年三星旗舰活动中亮相的设备的信息。就在今天早些时候，我们看了一下[三星 Galaxy Z Fold 2 及其打孔显示屏](https://www.xda-developers.com/samsung-galaxy-z-fold-2-leak-blurry-photo-showing-off-hole-punch-display/)。现在，三星 Galaxy Buds Live 的新渲染图已经泄露，从各个角度展示了即将推出的耳塞，包括翼尖。

这些新的渲染图由泄密者*伊山·阿加瓦尔*提供给我们[。虽然我们已经在过去的几个渲染中看到了 Galaxy Buds(像](https://twitter.com/ishanagarwal24/status/1288405770405580800)[这个](https://www.xda-developers.com/samsung-galaxy-buds-live-bean-shaped-tws-earbuds-leak-three-colors-case/)、[这个](https://www.xda-developers.com/samsung-galaxy-note-20-galaxy-buds-live-leaks/)和[这个](https://www.xda-developers.com/samsung-galaxy-buds-live-active-noise-cancellation-play-store/))，但我们总是想知道这些设备实际上如何留在耳朵里。这些新的渲染提供了一个合理的解释，因为它们给了我们迄今为止对翼尖突起的最佳观察。

从侧面观看三星 Galaxy Buds Live 展示了芸豆形耳塞上的翼尖。扬声器格栅位于耳塞的下半部分，翼尖位于顶部。翼尖应该钩在耳廓上，有助于保持耳垂的位置。这些渲染也证实了即将到来的耳塞将不会有带硅尖的入耳式设计，扬声器将放在你的耳朵外面。

Galaxy Buds Live 是下周三星发布会上预计的五款设备之一。这些耳塞将最终为三星的生活方式 TWS 系列带来主动噪音消除支持。即将到来的耳塞的紧凑设计将更加贴近用户的耳朵，不会突出太多。这款耳塞的尺寸预计只有 2.8 厘米，使用起来会更舒适、更方便，尤其是对耳朵小的人来说。然而，新的设计可能会对隔音产生一些不利影响，因为耳塞没有任何硅胶组件来确保正确的密封。我们很快就会知道了。
