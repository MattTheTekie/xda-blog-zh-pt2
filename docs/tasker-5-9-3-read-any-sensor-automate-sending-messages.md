# Tasker 5.9.3 可以让你读取手机上的任何传感器，用任何应用程序自动发送消息，等等

> 原文：<https://www.xda-developers.com/tasker-5-9-3-read-any-sensor-automate-sending-messages/>

当谈到 Android 上的自动化应用程序时，Tasker 被广泛认为是最佳选择之一。该应用程序允许你创建基于特定条件自动触发的配置文件来执行任务，它提供了各种各样的插件，可以帮助你在 Android 设备上自动化几乎任何事情。更好的是，该应用程序定期更新，开发者不断发布新功能，以帮助您使用 Tasker 实现更多功能。在过去的几个月里，Tasker 在 beta 频道上收到了几个[重大更新](https://www.xda-developers.com/tasker-5-9-3-beta-5-customize-all-settings-automatically-freeze-apps/)，引入了新功能，如 [WhatsApp 呼叫自动化](https://www.xda-developers.com/tasker-5-9-3-beta-4-contact-via-app-wireguard-integration/)、[请勿打扰定制、传感器动作](https://www.xda-developers.com/tasker-5-9-3-beta-do-not-disturb-customization-sensor-actions/)等等。现在，开发人员正在推出该应用程序的重大更新，将这些功能的大部分引入稳定的版本。

**[XDA Tasker 提示&招数论坛](https://forum.xda-developers.com/u/tasker-tips-tricks)**

根据开发人员 joo Dias 最近的一篇博客文章，Tasker v5.9.3 已经开始在稳定频道上向用户推出，它包括了许多有趣的新功能。以下是 Tasker 最新更新中一些最值得注意的变化:

## Tasker 现在可以读取你手机上的任何传感器

此次更新为该应用程序带来了 2 个新条件和 2 个动作，将帮助您利用手机上的任何传感器。这项功能最初是在今年 2 月的测试版中提供的，随着最新的更新，最终对所有用户开放。由于该功能，Tasker 现在可以扫描您的设备的传感器，并允许您根据传感器状态自动执行任务。

任何传感器事件和状态将允许您监控传感器并对任何变化做出反应，测试传感器操作将从您的智能手机的传感器获取当前值，以帮助您在任务中使用它们，传感器信息操作将允许您获取传感器本身的信息，包括名称、功耗、最大范围等信息。有了这个功能，你就可以在双击设备时打开手电筒。

Tasker 现在还可以让你通过任何第三方应用程序，如 WhatsApp 或 Telegram，自动发送短信或打电话。查看以下链接的视频，了解该功能的实际应用:

## 完全免打扰控制

完整的免打扰定制是我们今年早些时候在 Tasker 的测试版更新中涉及的另一个功能，现在该功能终于进入了稳定版本。

通过该功能，您可以更改 Tasker 中的任何“请勿打扰”设置，并自定义诸如谁可以给您发消息或打电话、允许在屏幕上显示什么通知等内容。

## 通过链接触发 Tasker

更新还包括一个漂亮的新功能，让你通过链接触发 Tasker 中的事件。正如你在所附视频中看到的，你现在只需打开一个兼容的 URL(比如 tasker://secondary？text=hello&other=hi)在手机上。

除了上面提到的功能，Tasker 的最新更新还包括其他几个变化，您可以在下面的 changelog 部分找到这些变化。如果您有兴趣在您的设备上尝试这些功能，您可以从下面的 Play Store 链接下载更新，或者通过下面的 source 链接下载最新的 APK。
