# 谷歌记录器，但不是屏幕的注意力将来到老像素手机

> 原文：<https://www.xda-developers.com/pixel-4-recorder-app-screen-attention-older-pixel/>

**更新(美国东部时间 12/5/19 @ 11:00am):**正如承诺的那样，谷歌在 Pixel 4 上推出的 Recorder 应用程序现在可以用于旧款 Pixel 手机。

现在，围绕 Pixel 4 的所有最初宣传都已经平息，人们开始了解这款设备及其软件的细节。除了相机软件和流畅的 Android 体验的显而易见的候选产品，人们可以看到 Pixel 4 不仅仅具有旗舰功能。

[屏幕关注](https://www.xda-developers.com/android-q-smart-stay-screen-attention/)是 Pixel 4 首次推出的一项功能。它通过使用前置摄像头而不是 Soli 雷达传感器来保持屏幕处于打开状态。这一点实际上非常重要。虽然 Pixel 4 和 Pixel 4 XL 是唯一采用 [Soli 芯片](https://www.xda-developers.com/google-pixel-4-aware-music-control-gestures-android-q/)的设备，但基本上所有手机都有前置摄像头。这意味着屏幕注意力可以转移到其他设备上，因为不需要特殊的硬件。

然而，不幸的是，谷歌已经否决了任何希望屏幕注意来其他设备很快。当 *[和](https://www.androidpolice.com/2019/10/30/google-pixel-4-screen-attention-older-phones/)* 联系上谷歌时，谷歌表示它甚至没有打算将这一特性应用到更老的 Pixel 设备上。但有一点需要说明的是，他们明确表示:“*目前为“*”。这意味着，如果公司从消费者那里获得足够的兴趣，他们可能会改变主意。

但对于其他一些功能，未来看起来更光明。Redditor [/u/Valendr0s](https://www.reddit.com/r/GooglePixel/comments/dp7kyu/voice_recorder_transcription_app/f5xp1kd/?context=3) 在谷歌 Pixel 子编辑中询问 Pixel 4 的记录器应用程序是否会被带到较旧的 Pixel 设备上。谷歌官方回复说，他们计划在未来的软件更新中推出这款应用。没有具体的时间表，也没有具体的型号。

尽管如此，承诺是存在的。正如 Valendr0s 在他的帖子中指出的那样，Recorder 应用程序及其转录功能对于那些无法写笔记的残疾人来说非常有用。即使对于我们这些懒得去听完整的录音，而更喜欢有一段文字的人来说，这款应用的重要性也不能低估。

同样值得注意的是，另一位用户在评论该帖子时说，当他们侧装 [APK](https://www.apkmirror.com/apk/google-inc/google-recorder/google-recorder-1-0-271580629-release/google-recorder-1-0-271580629-android-apk-download/) 时，该应用程序在他们的 Pixel 3 XL 上工作。我们不能 100%确定这是真的，或者它是否也适用于非像素设备。话虽如此，如果你真的需要这款应用，还是值得看看这条路线。

## 更新:适用于像素 3、3a、2

上个月，谷歌表示，他们将在未来的软件更新中把 Pixel 4 的记录器应用程序带到旧的 Pixel 设备上。该应用程序的 1.1 版本增加了对 Pixel 2、Pixel 2 XL、Pixel 3、Pixel 3 XL、Pixel 3a 和 Pixel 3a XL 的支持。不幸的是，最初的谷歌 Pixel 和 Pixel XL 似乎没有包括在内(尽管这并不妨碍你从[侧面装载 APK](https://www.apkmirror.com/apk/google-inc/google-recorder/google-recorder-1-1-283481728-release/) )。如果你有一个旧的 Pixel 设备，你现在应该可以从 Play Store 安装 Recorder 应用程序了。
