# 小米米 Play 的内核源代码现已上市

小米试图在发布源代码方面做得更好。小米 Mi Play 的内核源代码现在可以在该公司的 GitHub 页面上找到。

去年年底，小米在中国推出了一款名为[小米 Play](https://forum.xda-developers.com/mi-play) 的新智能手机。设备[在推出](https://www.xda-developers.com/xiaomi-play-design-specifications-tenaa/)之前已经被取笑了一段时间，它受到了很多关注，因为它是该公司第一款带有摄像头和传感器水滴凹槽的智能手机。它配备了联发科 P35 SoC，4GB 内存，64GB 内部存储，以及双 12MP (f/2.2) + 2MP 后置摄像头设置。它的硬件和设计符合该公司新的 Redmi 子品牌。

[**小米米玩 XDA 论坛**](https://forum.xda-developers.com/mi-play)

当涉及到发布他们设备的内核源代码时，小米有一个古怪的历史。他们承诺在产品发布后的 3 个月内发布内核源代码，但去年他们未能在 Mi Pad 4 上实现这一目标。不过值得庆幸的是，他们在小米 Mi Play 上做得更好，你可以在下面链接的 GitHub 页面上找到该设备的源代码。
