# 一加 6T 的显示指纹扫描仪随着时间的推移越来越好

> 原文：<https://www.xda-developers.com/oneplus-6t-in-display-fingerprint-scanner-gets-better/>

一加 6T 最值得注意的特点之一是显示指纹扫描仪。这是一项萌芽技术，我们还没有在许多主流设备上看到。一加称之为“屏幕解锁”，他们最近在论坛上解释了如何随着时间的推移变得更好。

当一加 6T 首次宣布时，一加[解释了他们放弃传统指纹扫描仪而采用显示指纹扫描仪的决定](https://www.xda-developers.com/oneplus-explains-the-story-of-screen-unlock-on-the-oneplus-6t/)。他们开发这一功能已有一年多时间，并认为它已准备就绪。今天，他们在谈论你越用这个功能，它就能越好。

耶鲁大学的软件工程师刘说:

*“每次用户用指纹成功解锁手机时，传感器还会记录指纹初始设置阶段未注册的手指区域。这些指纹数据随后被添加到手机已经保存的数据中，使其更加完整。因此，每次使用显示器内指纹传感器时，它都会收集更多关于用户指纹的数据。这意味着随着时间的推移，手机解锁的速度会更快。”*

[**一加 6T XDA 论坛**](https://forum.xda-developers.com/oneplus-6t)

所以基本上，你用屏幕解锁越多，你的手指就学得越多。随着每次成功解锁，一加 6T 收集更多关于你的指纹信息。正如他们在之前的帖子中提到的那样，指纹数据存储在骁龙 845 芯片组的“信任区”中。

耶鲁还提到，屏幕解锁足够智能，可以在有小的不规则时识别指纹，例如水或切口。像以前一样，它会记录新的信息作为你的指纹档案的一部分。一加 6T 会像你了解手背一样了解你的指纹。
