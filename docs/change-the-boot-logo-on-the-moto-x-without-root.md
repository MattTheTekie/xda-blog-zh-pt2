# 把 Moto X 上的开机 Logo 换成无根！

> 原文：<https://www.xda-developers.com/change-the-boot-logo-on-the-moto-x-without-root/>

你知道启动动画之前出现的闪屏启动标志吗？嗯，定制的徽标只是一个位图图像，存储在一个大多数用户永远不会涉足的位置。XDA 精英意识到开发者 [jcase](http://forum.xda-developers.com/member.php?u=2376614) 找到了负责的文件，事实证明，你甚至不需要 root 权限来修改它。

Jcase 分享了备份默认映像、“取消定制”bootloader 映像和刷新新徽标所需的简单说明。新徽标必须采用 BMP 格式，并且大小与原始图像相同，即 720 x 1280。

请注意，要备份当前徽标，您确实需要 root 访问权限。然而，刷新一个新的，是一个简单的快速启动命令，不需要你的设备是根。

要开始，只需前往[原始线程](http://forum.xda-developers.com/showthread.php?t=2445150)。如果你是 boot 用户，你应该备份引导标识，以防万一你想要回来。
