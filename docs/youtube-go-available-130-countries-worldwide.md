# YouTube Go 现已在全球 130 多个国家推出

> 原文：<https://www.xda-developers.com/youtube-go-available-130-countries-worldwide/>

2016 年 9 月，谷歌在印度发布了 YouTube Go [预览版，](https://www.xda-developers.com/google-announces-youtube-go-in-india-the-data-friendly-way-to-youtube/)这是一款为发展中市场用户设计的轻量级 YouTube 应用。自那以后，该公司将这款应用程序带到了另外 14 个国家，包括印度尼西亚、尼日利亚和泰国，并于本周宣布，YouTube Go 可在全球 130 多个国家使用。

YouTube 的产品经理 Jy Akkad 在一篇博客文章中写道:“我们一直在努力开发一款为你服务的 YouTube 应用程序，它的全新功能是根据你的偏好、愿望和需求而设计的。”。“我们迫不及待地想看看你可以利用 YouTube 的力量做些什么。”

YouTube Go 是谷歌在数据价格昂贵或无法广泛获取的市场中改善移动 YouTube 体验的努力的高潮。该应用程序离线工作，没有连接，并让用户控制每个视频的移动数据消耗。主要功能包括趋势视频中心，视频预览，以及下载任何视频而不是流媒体的选项。

本周，谷歌为 YouTube Go 增加了更多功能。这款应用现在可以让你下载、播放和分享高质量的视频，这在以前是不可能的。(除了基本和标准质量分辨率之外，还提供高质量选项。)

此外，你可以通过在应用程序的主屏幕上向下滑动来访问新的个性化内容，当新视频上传到你关注的频道时，应用程序会通知你。分享功能也得到了改进——谷歌表示，它使“分享附近”功能变得更加容易，这可以让你通过蓝牙和 Wi-Fi 直接向 YouTube 用户发送视频。现在可以同时分享多个视频。

谷歌最近推出了一系列“Go”品牌的应用，包括 Google Go、 [Files Go](https://www.xda-developers.com/google-files-go-update-tablets-sd-card/) 等等。这是该公司 Android Go 计划的一部分- [我们已经写了 Android Go 如何帮助旧手机运行 Android 8.1 Oreo](https://www.xda-developers.com/android-go-old-android-8-1-oreo/) ，以及即将推出的 [Android Go 手机](https://www.xda-developers.com/micromax-india-first-android-go-phone-bharat-go/)。
