# 首款官方 Galaxy Note II N7100 固件现已上线

> 原文：<https://www.xda-developers.com/first-official-galaxy-note-ii-n7100-firmwares-now-online/>

毫无疑问，三星 Galaxy Note II 是一头野兽。这东西在任何方面都是巨大的。无论是屏幕尺寸、内部存储，还是 RAM 没有另一个设备符合标准。这似乎也延伸到了它的官方固件，重达 1043 兆字节。我怎么知道这个？因为 XDA 认识到开发者 [Faryaab](http://forum.xda-developers.com/member.php?u=1828342) 非常友好，最近更新了他的官方固件线程，为 Note II 提供了一个基于欧洲 4.1.1 的 ROM 的链接。

这一特别的更新似乎引起了相当大的兴奋，因为它带来了“多窗口”功能，这是一些设备所有者以前无法获得的。当然，你不需要真的拥有这个设备来下载并拆开它。也许这个相当漂亮的功能甚至可以应用到其他三星设备上(i9100 first please 、nudge nudge、wink wink)...).那里可能还有很多其他值得一试的东西。至少人们希望如此大小的 ROM。

如果你想升级到这个最新的固件，可以通过 Odin 刷新。你应该知道，如果你来自欧洲以外的地区，这将改变你的 CSC。前往[股票固件线程](http://forum.xda-developers.com/showthread.php?t=1875311)开始。
