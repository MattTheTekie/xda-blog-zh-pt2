# OnePlus 3 和 OnePlus 3T 获得第二个 Android Pie 测试版

> 原文：<https://www.xda-developers.com/oneplus-android-pie-beta-oneplus-3-3t/>

一加为其用户保持了一个活跃的测试版更新渠道，定期为其智能手机测试 OxygenOS 的新功能。虽然相对较新的 OnePlus 5/5T 和 OnePlus 6/6T 在更新方面赢得了公司的大部分注意力，但公司没有忘记更老的型号- OnePlus 3 和 OnePlus 3T。在[保证](https://www.xda-developers.com/oneplus-3-oneplus-3t-android-p-update/)近一年后，[公司最近宣布了 OnePlus 3/3T](https://www.xda-developers.com/oneplus-3-oneplus-3ts-oxygenos-beta-android-9-pie/) 的 Android 9.0 Pie，作为社区测试版的一部分，在推出稳定版本之前清除 bug 或不准确之处。随着测试的进行，该公司现在为两款基于 Android Pie 的智能手机带来了第二次测试更新。

[**一加 3 XDA 论坛**](https://forum.xda-developers.com/oneplus-3</span><span%20>) [**一加 3T XDA 论坛**](https://forum.xda-developers.com/oneplus-3t)

随着最新社区测试版的更新，OnePlus 3 和 3T 对一些系统应用程序的工作进行了一些改进，如消息和时钟，以及对有线耳机的更好支持。OnePlus 3 和 OnePlus 3T 的变更日志说明修复了以下错误:

*   时钟计时器和秒表界面显示不完整
*   插入耳机时，麦克风不工作
*   本机消息应用程序的崩溃问题

请注意，社区测试版是**而不是**与[公开测试版更新](https://www.xda-developers.com/oxygenos-open-beta-30-28-oneplus-5-5t-april-2019-security-patch-parking-location-feature/)相同，一加将其推送到其智能手机上，以在稳定版本中实施之前测试最新功能。

因此，如果你有一台 OnePlus 3 或 OnePlus 3T 运行稳定的 OxygenOS 版本，并希望尝试 Android Pie 更新，请使用下面的链接为你的设备下载固件包，并将它们放在存储的根目录中。你可以进入设置>系统更新，然后从右上角的菜单中选择“本地更新”来安装更新。

或者，您可以通过同时按住音量增大和电源按钮以快速启动模式重新启动，使用音量按钮进入恢复模式，然后选择“从内部存储安装”来刷新更新。
