# 在 ChromeOS 上启用 ADB 和 Fastboot 的指南

> 原文：<https://www.xda-developers.com/guide-to-enable-adb-and-fastboot-on-chromeos/>

[亚行](http://www.xda-developers.com/android/lets-talk-about-adb/)和[f](http://www.xda-developers.com/android/easily-set-up-adb-and-fastboot-on-a-mac/)[astboot](http://www.xda-developers.com/android/easily-set-up-adb-and-fastboot-on-a-mac/)无论您使用的是 Windows、Linux 还是 OSX，设置好其中任何一个并开始修改您的设备通常都不会太难。不过，有一件事我之前没有见过，直到现在，那就是如何在谷歌越来越受欢迎的 Chrome 操作系统或其开源兄弟 Chrome 操作系统上设置 ADB 和 fastboot 。

Chrome OS 和搭载它的 ChromeBooks 曾经被一些人认为只不过是一个盒子里的网络浏览器，现在变得越来越受欢迎。因此，许多人现在似乎很满足于用这些高度便携且价格实惠的设备来取代他们更大、功能更强的笔记本。当然，不利的一面是，一个精简的操作系统意味着更少的灵活性和功能。对于那些无法在 20 分钟内不摆弄移动设备的人来说，这可能会让 ChromeOS 看起来像是一个更可行的移动选择。

XDA 公认的贡献者 [Quinny899](http://forum.xda-developers.com/member.php?u=3563640) 写了一个指南，涵盖了在 Chrome 和 Cromium OS 上启用 ADB 和 fastboot 命令所需的步骤。步骤不多，过程相当简单。下载适当的文件并输入一些终端命令后，您应该可以开始了。你需要一台 ChromeBook(或者运行 Chrome/Chromium 的东西)和一点 Linux 知识，或者能够复制和粘贴终端命令。不过，如果事情并不总是按照计划进行，那么一点点知识肯定会派上用场。当下载这些文件的时候，你也可以选择一杯热饮。

如果你正在运行 Chrome 或 Chromium OS，并想亲自尝试一下，请查看[教程线程](http://forum.xda-developers.com/showthread.php?p=38232908)了解更多信息。
