Wear OS应用程序现在允许您添加5个以上的磁贴

原文：https://www.xda-developers.com/wear-os-app-add-more-than-5-tiles/#0001-01-01

早在去年5月，谷歌推出了一项名为“Tiles”的穿戴操作系统的新功能。该功能旨在帮助用户只需轻扫即可访问Wear OS智能手表上最常见的操作。在推出时，用户只能访问5个标题，包括目标、下一个事件、预测、心率、标题和计时器。但是第三方开发者很快设法发布了一个非官方的Tiles API 来给配套的应用添加定制的Tiles。

此后不久，谷歌更新了Wear OS应用程序，增加了一个菜单，帮助用户轻松管理磁贴，并根据他们的喜好排列它们。然而，即使有了这些新功能，用户也只能在智能手表上添加5个磁贴。现在，谷歌推出了Wear OS应用程序的更新，解决了这个问题。

有了Wear OS app版，你终于可以给智能手表添加5个以上的磁贴了。正如你在附上的截图中看到的，我们设法在应用程序上添加了10个磁贴，包括TicHealth、Forecast、Next event、TicPulse、Timer、简单计算器、Headlines、Hand-Wash Timer、Voicememo和TicExercise。但我们不确定该应用程序现在是否仅限于10个瓷砖，因为我们用完了更多的瓷砖来添加。

如果你拥有一款Wear OS智能手表，并希望尝试这项新功能，你可以从下面的Play Store链接下载最新版本的应用程序。或者，如果你还没有收到Play Store的更新，你可以从APKMirror下载最新的APK。如果您可以访问超过10个磁贴，请在下面的评论部分告诉我们您可以向应用程序添加多少个磁贴。
