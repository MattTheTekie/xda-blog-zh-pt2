# 小米 Poco F1 和小米米 8/探索者版内核源代码现已上市

小米在同一个内核源代码分支下发布了 Poco F1、Mi 8 和 Mi 8 Explorer 版的内核源代码。

钢蓝色

最近，小米在为其手机发布内核源代码方面一直在改善记录。今年早些时候，该公司承诺在设备发布后的三个月内发布内核源代码。虽然这并不理想，但确实是一个受欢迎的改进。[小米花了很长时间才发布小米 Mi A1 等手机的内核源代码](https://www.xda-developers.com/xiaomi-mi-a1-android-oreo-kernel-source-code/)，但对于更新的设备版本，该公司基本上按时发布了源代码。

小米在发布后不久就发布了小米 Mix 2S 和小米 [Mi 8 SE](https://www.xda-developers.com/xiaomi-mi-8-se-kernel-source-code/) 的内核源代码。一周前，该公司的新 Poco 品牌在印度推出了小米 Poco F1，并公开承诺在发布当天发布该设备的内核源代码。Poco 还表示，[解锁引导程序不会使 Poco F1](https://www.xda-developers.com/xiaomi-poco-f1-developer-community/) 的保修失效，因为这款手机旨在方便开发者。

几天前，小米发布了小米 Mi A2 (jasmine-sprout)和小米 Mi A2 Lite (daisy-sprout)的内核源代码，这本身就比 Mi A1 的内核源代码发布在时间上有了明显的改善。现在，该公司已经兑现了承诺，发布了 Poco F1 的内核源代码。三个设备的内核源代码分支是相同的:小米 Mi 8(设备代码名:迪普)、小米 Mi 8 探索版(设备代码名:ursa)、Poco F1(设备代码名:铍)。

这三种设备的内核源代码的发布意味着开发社区现在可以开始开发 TWRP、定制 rom 和内核等。Poco F1 特别实惠的起价肯定会使它在 XDA 的开发中成为一款非常受欢迎的设备，其内核源代码的及时发布对于已经非常适合狂热用户的用户来说是锦上添花。
