  3. 在您的电脑上下载并安装 Microsoft Visual C++ 2008可再发行软件包(x86) 和 Microsoft Visual C++ 2010可再发行软件包(x86) 。这些是SamFirm下载您的固件所必需的。

  6. 打开SamFirm。一旦它打开，确保您选择了该区域下的自动选项。这将使下载文件自动进行，所以你不需要每次下载一个内部版本号。
