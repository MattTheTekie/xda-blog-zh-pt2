安装很容易，即使你以前从未使用过 AROMA。在自定义恢复中闪一下拉链，AROMA 就开了。选择要在设备上安装的音乐和视频播放器。安装程序将安装它们，然后重新启动。如果你想快速安装你最喜欢的媒体应用程序，这绝对是值得一试的。请注意，所提供的媒体播放器都是试用版。
