# 在任何安卓手机上获得谷歌 Pixel 3 的 Flip to Shhh 功能

> 原文：<https://www.xda-developers.com/google-pixel-3-flip-to-shhh-tasker/>

谷歌 Pixel 3 和 Pixel 3 XL 于上周正式发布。虽然有些人可能会认为他们不值得从以前的 Pixel 设备升级，因为硬件没有太大变化，但我们知道[谷歌确实在软件上做得对](https://www.xda-developers.com/google-pixel-3-xl-camera-software-design-pixel-stand/)。你可能已经知道了，谷歌给它们添加了很多独家功能。其中一个是[“翻转到 Shhh”，](https://www.xda-developers.com/digital-wellbeing-flip-to-silence-google-pixel-2/)，当你把像素面朝下放在桌子或其他任何表面上时，它可以让你将像素置于勿扰模式。幸运的是，几乎没有什么东西是特定设备的专利。如果该特性很好，开发社区会将其移植到其他 Android 设备上。

Android 的精髓之一是能够自动化我们设备上的一切。Tasker 是迄今为止最强大和最受欢迎的应用程序，可以让你的手机随时按你的意愿工作。这就是我们将在本文中用来在几乎任何 Android 设备上复制“翻转嘘”功能的东西。老实说，我在棉花糖时代就有这个功能，在它变得很酷之前。
