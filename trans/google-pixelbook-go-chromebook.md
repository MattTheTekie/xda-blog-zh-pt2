# Pixelbook Go 是谷歌在高端 Chromebook 上的最新尝试

> 原文：<https://www.xda-developers.com/google-pixelbook-go-chromebook/>

速度、简单和安全:这是谷歌 Chrome OS 的座右铭，这是一款基于 Linux 的笔记本电脑操作系统。得益于 Chrome OS 的低系统要求，谷歌在教育市场获得了一席之地，导致廉价 Chromebooks 在学生中激增。然而，谷歌一直在努力扩大 Chrome OS 在这个市场以外的吸引力。他们希望通过 Pixelbook Go 改变这一点。

凭借 Android 和 Linux 应用支持等功能以及 Pixelbook 等高端硬件，谷歌试图将 Chrome OS 作为企业用户的可行替代产品进行营销。虽然 Pixel book[受到了评论家的好评](https://www.xda-developers.com/google-pixelbook-great-experience/)，但它的继任者，可拆卸的 Pixel Slate 平板电脑，[却遭受了硬件不足和软件优化不佳的](https://www.xda-developers.com/google-pixel-slate-video-review/)。不过，谷歌并没有放弃高端 Chromebooks，正如他们在 Pixelbook Go 上展示的那样。

谷歌的新款 Pixelbook Go 将采用更传统的笔记本电脑外形。你不能翻转设备以在帐篷模式下使用它，也不能拆卸屏幕以将其用作平板电脑。这是一台纯粹的笔记本电脑。谷歌为希望在旅途中工作的消费者设计了这款笔记本电脑(因此名称中有“go”)，因此 Pixelbook Go 的背面有一个粗糙的棱纹纹理，便于放在腿上或手中。笔记本电脑的顶部也有一个光滑的哑光表面，所以当你拿在手中时，也会增加握持感。

由于 Pixelbook Go 非常紧凑，所以它缺少一个全尺寸的 USB 端口，但它有两个 USB Type-C 端口，每侧一个，并伴有充电 led。令人惊讶的是，左侧还有一个 3.5 毫米耳机插孔，这是 Pixel 4 智能手机所没有的。当笔记本电脑打开时，你会看到显示器上方有一个 200 万像素的摄像头(高达 1080p@60fps 视频)，两个前置扬声器，两个用于谷歌助手热门词汇检测的远场麦克风，一个 13.3 英寸(16:9)显示屏，一个不带数字小键盘的标准笔记本电脑键盘和触控板。键盘左下角有一个谷歌助手键，但在其他方面与大多数其他 Chromebooks 相似。

奇怪的是，谷歌已经决定放弃指纹扫描仪和 Pixelbook 笔的支持。也许他们看到没有多少人使用这两个功能，或者他们无法在不增加笔记本体积的情况下增加支持。

谷歌去年在 Pixel Slate 上犯了一个错误，在入门级型号上提供了[英特尔赛扬处理器，但似乎他们已经从这个错误中吸取了教训，因为 Pixelbook Go 一开始就配备了英特尔酷睿 M3 和 8GB RAM。其他配置包括 16GB 内存和 256GB 存储的英特尔酷睿 i7，以及 FHD 或 4K 分辨率。所有型号都配备了谷歌的 Titan C 安全芯片，以防止恶意攻击者试图加载被篡改或过时的软件。](https://www.xda-developers.com/google-stops-selling-entry-level-pixel-slate-celeron/)

你可以购买“黑色”或“非粉色”的 Pixelbook Go，起价 649 美元。美国和加拿大今天开放预购，英国将于明年 1 月开始。
