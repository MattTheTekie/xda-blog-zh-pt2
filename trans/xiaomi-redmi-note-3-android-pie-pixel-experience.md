# 小米 Redmi Note 3 基于安卓派获得像素体验

自从 Android Pie 推出以来，我们已经看到了许多端口的发布，XDA 高级成员 Carlos Arriaga 分享了小米 Redmi Note 3 的像素体验端口。

Redmi Note 3 是第一批真正吸引我眼球的小米智能手机之一。它没有最快的硬件，MIUI OEM ROM 也不太适合我，但那个带定制 ROM 的设备价格惊人。从那以后，我对常规的 Redmi 系列和他们的 Redmi Note 系列印象深刻，并很高兴看到开发人员社区围绕它蓬勃发展。[自从 Android 9 Pie source 发布以来](https://www.xda-developers.com/android-pie-source-code-aosp/)我们已经看到了许多端口的发布，就在昨天 XDA 资深会员 [Carlos Arriaga](https://forum.xda-developers.com/member.php?u=6331811) 分享了他们的 Pixel 体验端口。

自定义 ROM 基于 Android 9 派。正如你所期望的，大多数硬件都工作正常，所以对于一些人来说，它可能值得作为日常驱动程序使用。以下是目前为止我们所知道的小米 Redmi Note 3 像素体验 ROM 的列表:

什么在起作用:

*   无线局域网（wireless fidelity 的缩写）
*   RIL
*   移动数据
*   全球（卫星）定位系统
*   照相机
*   手电筒
*   便携式摄录一体机
*   蓝牙
*   调频电台
*   指纹读取器
*   红外辐射（Infrared Radiation）
*   光
*   声音/振动

已知问题:

*   硬件加密
*   Selinux 强制执行

开发人员告诉我们，没有任何理由刷新 Gapps，因为它们已经包含在内，但是有一个特殊的文件需要刷新。该线程说“无需重置配置”就可以刷新它，链接会将你带到一个电报片段下载，其中有一个标签，看起来可以修复 Android 9 Pie ROMs 上的 VoLTE。
